# E-commerce React App

A modern e-commerce web application built with React, featuring user authentication, product browsing, shopping cart management, and checkout functionality.

## Features

- User Login & Signup
- Product Listing
- Product Details Page
- Shopping Cart
- Checkout Page

## Tech Stack

- React
- JavaScript
- CSS
- Vite
